# Reference Screenshots — Zero Fasting App

Screenshots taken from the Zero Fasting app (iOS) as visual reference for Aeterna.

## Main Screen (Today)

### Layout from top to bottom:

1. **Header**
   - Green check icon with "2" badge (top left)
   - "Zero" logo centered (red/burgundy text)
   - "+" button (top right)

2. **Weekly Calendar**
   - Horizontal row: WED THU FRI SAT SUN MON TUE
   - Below each day: small circle
   - Active day (MON): filled green circle
   - Days without activity: empty grey circles

3. **Timer Card** (white background, rounded corners, soft shadow)
   - Circular progress ring (light grey track, subtle orange progress)
   - The ring has a thicker outer border and a thinner inner one
   - Centered text inside the ring:
     - "SINCE LAST FAST" (grey, small caps, top)
     - "00:44:11" (black, large monospace tabular font)
     - "EDIT 13H GOAL" (red/burgundy, clickable link, bottom)

4. **Start Fasting Button**
   - Full width inside the card
   - Burgundy/wine color (#7B2D4B approx.)
   - White "Start Fasting" text
   - Rounded corners

5. **Protein Score Card** (below the timer)
   - Semicircular gauge (orange gradient, left to right)
   - Grey background track
   - Orange arc reaches ~40% of the path
   - Large "40" number centered below the arc
   - "Protein Score" label below the number
   - (i) icon at top right of the card

6. **Bottom Tab Bar**
   - 3 tabs:
     - "Today" (red circle icon, active)
     - "Explore" (mountain/arrow icon)
     - "Me" (person icon)
   - White background
   - Active tab highlighted with background pill

## Design Observations

- The Zero app uses **light mode** (white/light grey background)
- Aeterna will use **dark mode** — invert the palette
- Cards have generous rounded corners (~16px)
- The timer dominates ~60% of the screen
- The protein score is secondary but visible without scrolling
- Clean typography, lots of white space
- The primary action button is large and high-contrast
